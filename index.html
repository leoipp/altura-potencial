Convenção de nomeclaturas para variáveis.

As variaveis seguem as nomeclaturas a seguir. mapbiomas: Classes de formação de florestas naturais filtradas nos passos anterior, 3 classes presentes (formação floresta, campestre e savânica); count: Número de pulsos emitidos à superficie terrestre pelo sistema LiDar; rh100: Altura relativa de copa acima do solo; tempMA: Temperatura média anual; precMA: Precipitação média anual; NAnuvens: Número de dias sem cobertura de nuvens; Inf100: Número de meses com precipitação inferior a 100 milímetros; mde: Modelo digital de elevação; decliv: Declividade

Modelagem por Machine Learning Random Forest

Seleção dos inputs do modelo que apresentarem melhores correlações com a variavel rh100 (). E seleção do output de predição do modelo, no caso, esperamos estimar a Altura Relativa dos povoamentos florestais.

input <- Altura_potencial %>%
    select(-mapbiomas) %>%
    select(-mde) %>%
    select(-tempMA) %>%
    select(-count) %>%
    select(-rh100)
    
output <- Altura_potencial %>% select(rh100)

Treinamento do modelo Random Forest

O modelo será submetido a validação cruzada de 5K folds para melhor acurácia.

drivers <- cbind(output, input)
                        
folds <- drivers %>%
    row.names() %>%
    groupKFold(k = 5) 
                        
group_fit_control <- trainControl(index = folds, method = "cv", number = 5)
                        
                        
rf.altura <- train(rh100 ~ .,                                    
                    data = drivers,
                    method='rf',                                     
                    importance=T,
                    trControl = group_fit_control)

rm(folds, group_fit_control, drivers)

Configurando a camada declividade como referencia

Para os proximos raster apresentarem matrizes de mesma configuração, utilizaremos a camada declividade como referencia.

decliv <- abs(raster('decliv.tif'))
decliv <- setMinMax(decliv)
ref <- abs(decliv)

Configurando demais camadas

NAnuvens <- raster('NAnuvens.tif') %>% crop(ref)
NAnuvens <- abs(NAnuvens)
NAnuvens <- raster(vals=values(NAnuvens),
                        ext=extent(ref),
                        nrows=dim(ref)[1],
                        ncols=dim(ref)[2])
NAnuvens <- setMinMax(NAnuvens)
                        
                        
Inf100 <- raster('Inf100.tif') %>% crop(ref)

Stack de camadas e predição

Com as camadas nas devidas dimensoes, podemos fazer o stack das mesmas para entao utilizar no modelo treinado. O objeto Raster_altura será o produto final desta análise.

layers <- stack(decliv, NAnuvens, Inf100, precMA)
names(layers) <- c("decliv","NAnuvens","Inf100","precMA")
                        
Raster_altura <- predict(layers, rf.altura)
